# A iOS 8 Navidrome Player

## About this project!
So i recently started using navidrome to stream music locally, i like to use a different device to stream music and i noticed that on my jailbroken iPod touch 5 i coudn't download any apps for navidrome so my next thought was to just use the website sadly it didn't load because its to "new", I decided to make a simple website for it to play the music via safari!

For background play on my jailbroken iPod Touch 5, iOS 8.4.1 i'm using [Watchdog Lite](https://www.ios-repo-updates.com/repository/bigboss/package/com.theronen.watchdoglite/) from [BigBoss Repo](https://www.ios-repo-updates.com/repository/bigboss/)
